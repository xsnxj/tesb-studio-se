cBean

cBean properties

Component Family

Processor

Function

cBean invokes a Java bean that is stored in the Code node of the Repository or registered by a cBeanRegister.

Purpose

cBean allows you to invoke a Java bean that is stored in the Code node of the Repository or registered by a cBeanRegister.

Basic settings

Reference

Select this option to reference a Java bean registered by a cBeanRegister.

In the Id field that appears, enter the Id of the Java bean.

 New Instance

Select this option to invoke a Java bean that is stored in the Code node of the Repository.

In the Bean class field that appears, enter the name of the bean class.

For more information about creating and using Java Beans, see Talend Open Studio for ESB User Guide.

 

Specify the method

Select this check box to enter the name of a method to be included in the bean.

Usage

cBean can be a start, middle or end component in a Route.

Limitation

 n/a
