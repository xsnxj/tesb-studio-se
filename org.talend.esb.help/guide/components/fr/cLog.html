cLog

Propriétés du cLog

Famille de composant

Miscellaneous

Fonction

Le composant cLog enregistre des échanges de messages dans le mécanisme sous-jacent de log. Apache Camel fournit le Regular logger et le Throughput logger. Le premier permet d'enregistrer les logs de tous les échanges de messages et le second permet de paramétrer le nombre de messages que vous souhaitez afficher. Par défaut, Apache Camel enregistre les logs de tous les échanges de messages.

Objectif

Le cLog est utilisé pour enregistrer des échanges de messages.

  Level Sélectionnez un niveau de log parmi DEBUG, ERROR, INFO, OFF, TRACE et WARN.
 

Use default output log message

Sélectionnez cette option pour utiliser le message de log de sortie par défaut fourni par le mécanisme sous-jacent d'enregistrement de logs.

 

Options / None

(Pour le message de log de sortie par défaut uniquement)

Sélectionnez cette option pour n'effectuer aucune action sur le message de log.
 

Options / Specifies a group size for throughput logging

(Pour le message de log de sortie par défaut uniquement)

Sélectionnez cette option pour utiliser le Logger Throughput et spécifier la taille du groupe de messages à afficher.

Size : Saisissez un nombre entier spécifiant la taille du groupe de messages à afficher.

 

Options / Group message stats by time interval (in millis)

(Pour le message de log de sortie par défaut uniquement)

Sélectionnez cette option pour utiliser le Logger Throughput et regrouper les statistiques des messages.

Interval : Les statistiques des messages seront regroupées durant l'intervalle de temps que vous spécifiez (en millisecondes).

Delay : Paramétrez le délai initial (en millisecondes) pour les statistiques des messages.

 

Options / Format the log output

(Pour le message de log de sortie par défaut uniquement)

Sélectionnez cette option afin de formater la sortie du log. Cliquez sur le bouton [+] autant de fois que nécessaire pour ajouter des arguments dans le tableau. Cliquez sur le champ value correspondant au nom de l'argument et saisissez une valeur. Consultez le site http://camel.apache.org/log.html (en anglais) pour les options disponibles.
 

Specify output log message

Sélectionnez cette option pour spécifier le message de log de sortie.

Message : Utilisez le langage Simple pour construire un message dynamique qui sera enregistré dans le log.

Utilisation

Le composant cLog est utilisé en tant que composant intermédiaire ou de fin dans une Route.

Limitation

 n/a

Scénario associé

Pour un scénario associé, consultez la section intitulée « Scénario : Router des messages selon un critère ».
