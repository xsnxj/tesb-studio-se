cJavaDSLProcessor

Propriétés du cJavaDSLProcessor

Famille de composant

Processor

Fonction

Le composant cJavaDSLProcessor implémente des producteurs et des consommateurs d'échanges de messages ou implémente un Message Translator utilisant le DSL Java (Domain Specific Language).

Objectif

Le cJavaDSLProcessor est utilisable pour injecter rapidement du code à l'aide du DSL Java. Si le code de la classe interne devient plus complexe, il est conseillé de le refactoriser dans une classe séparée.

Basic settings

Code

Saisissez le code que vous souhaitez implémenter à l'aide d'un DSL Java.

Utilisation

Le cJavaDSLProcessor peut être utilisé en tant que composant de début, composant intermédiaire ou composant de fin dans une Route.

Limitation

 n/a

Scénario associé

Pour un scénario associé, consultez la section intitulée « Scénario : Ecouter un message dans une Route ».
