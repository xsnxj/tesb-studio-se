cProcessor

Propriétés du cProcessor

Famille de composant

Processor

Fonction

Le composant cProcessor implémente des consommateurs d'échanges de messages ou implémente un Message Translator.

Objectif

Le cProcessor peut être utilisé pour remanier rapidement du code. Si le code dans la classe interne devient plus complexe, il est recommandé de le refactoriser dans une classe séparée.

Basic settings

Imports

Saisissez le code Java qui vous aident à importer, si nécessaire, les bibliothèques utilisées dans le champ Code.

 Code

Saisissez le code Java que vous souhaitez implémenter.

Utilisation

Le cProcessor peut être utilisé comme composant intermédiaire ou composant de fin dans une Route.

Limitation

 n/a
